JavaScript Cheat Sheet

Data Types

  • String - a sequence of characters
  • Number - a numeric value
  • Boolean - true or false
  • Undefined - a variable with no value
  • Null - a variable with a null value


Functions

  • function functionName() {} - defines a function
  • return value; - returns a value from a function
  • functionName(); - calls a function


Objects

  • var objectName = {}; - creates an empty object
  • objectName.propertyName = value; - adds a property to an object
  • objectName.methodName = function() {}; - adds a method to an object


Arrays

  • var arrayName = []; - creates an empty array
  • arrayName.push(value); - adds a value to the end of an array
  • arrayName.pop(); - removes the last value from an array


Control Structures

  • if (condition) {} - executes code if a condition is true
  • if (condition) {} else {} - executes one block of code if a condition is true and another if it is false
  • for (var i = 0; i < length; i++) {} - executes a block of code a specified number of times
