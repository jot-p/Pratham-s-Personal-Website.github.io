HTML Cheat Sheet

Basic HTML Tags

Here are some of the most common HTML tags:

  • <html> - defines the document as an HTML file
  • <head> - contains metadata about the document
  • <title> - defines the title of the document (displayed in the browser's title bar)
  • <body> - contains the visible content of the document
  • <h1> - defines a heading (there are also h2, h3, etc. for different levels of headings)
  • <p> - defines a paragraph
  • <a> - defines a hyperlink
  • <img> - displays an image



HTML Attributes

Attributes provide additional information about HTML elements. Here are some commonly used attributes:

  • class - defines a class for an element (used for styling with CSS)
  • id - defines a unique ID for an element
  • src - specifies the URL of an image or other media file
  • href - specifies the URL of the linked page
  • alt - provides alternative text for an image (useful for accessibility)



HTML Forms

Forms allow users to input data and submit it to a server. Here are some form-related tags:

  • <form> - defines a form
  • <input> - defines an input field
  • <textarea> - defines a multi-line input field
  • <select> - defines a drop-down list
  • <option> - defines an option in a drop-down list
  • <button> - defines a clickable button



HTML5 Elements

HTML5 introduced new elements that provide semantic meaning to content. Here are a few examples:

  • <header> - defines a header for a section
  • <nav> - defines a navigation menu
  • <section> - defines a section of content
  • <article> - defines an article or blog post
  • <footer> - defines a footer for a section
