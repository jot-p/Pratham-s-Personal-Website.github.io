CSS Cheat Sheet

CSS Syntax

CSS has the following syntax:

		selector {
			property: value;
			property: value;
			...
		}


CSS Selectors

Selectors are used to select HTML elements based on their name, id, class, attributes, and more. Here are some common selectors:

  • element selector: selects all elements of a certain type, e.g. "p" selects all paragraphs
  • id selector: selects an element with a specific id, e.g. "#header" selects the element with id="header"
  • class selector: selects elements with a specific class, e.g. ".highlight" selects all elements with class="highlight"
  • attribute selector: selects elements based on their attributes, e.g. "input[type='text']" selects all text input fields
  • descendant selector: selects elements that are descendants of another element, e.g. "ul li" selects all list items that are inside a unordered list


CSS Properties

CSS properties are used to style HTML elements. Here are some common properties:

  • color: sets the color of text
  • background-color: sets the background color of an element
  • font-size: sets the font size
  • font-family: sets the font family
  • text-align: sets the text alignment
  • margin: sets the margin around an element
  • padding: sets the padding inside an element
  • border: sets the border around an element


CSS Units

CSS uses different units to specify measurements. Here are some common units:

  • px: pixels
  • %: percentage
  • em: relative to the font-size of the element
  • rem: relative to the font-size of the root element
  • vh: viewport height
  • vw: viewport width
